# nvim-markdown-preview-lite

A minimal Markdown preview plugin for Neovim written in Lua.

## Features

- Split window preview
- Live update on save
- Zero dependencies (except pandoc)

## Requirements

- Neovim >= 0.7
- pandoc

## Installation (lazy.nvim)

```lua
{
  "momospc/nvim-markdown-preview-lite",
}
```

## Usage

```nvim
:MarkdownPreviewLite
```

## Help

```nvim
:help markdown-preview
```
